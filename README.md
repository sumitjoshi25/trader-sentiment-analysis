# Trader Performance vs Market Sentiment Analysis

## Objective
This project analyzes how market sentiment (Fear vs Greed) affects trader behavior and performance on the Hyperliquid platform.

## Datasets
1. Bitcoin Fear & Greed Index
2. Hyperliquid Historical Trader Data

## Methodology
1. Data cleaning and preprocessing
2. Dataset alignment by date
3. Feature engineering
4. Sentiment based analysis
5. Trader segmentation

## Key Insights
1. Traders generate higher average PnL during Greed markets.
2. Trade frequency increases during Greed periods.
3. Larger position sizes during Fear periods often lead to higher losses.

## Strategy Recommendations
1. Reduce leverage during Fear markets.
2. Increase trade frequency moderately during Greed markets.
3. Use sentiment indicators to adjust trading risk.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
